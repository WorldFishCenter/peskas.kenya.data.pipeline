# Get Total Catch Bounds using IQR method

Get Total Catch Bounds using IQR method

## Usage

``` r
get_total_catch_bounds_iqr(data = NULL, multiplier = 1.5)
```

## Arguments

- data:

  A data frame containing required columns

- multiplier:

  multiplier for IQR range (default is 1.5)

## Value

A data frame with upper bounds for each landing site and gear
combination

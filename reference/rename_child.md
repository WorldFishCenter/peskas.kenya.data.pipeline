# Rename Child Elements in Nested Kobotoolbox Data

This internal function renames child elements in nested Kobotoolbox data
structures.

## Usage

``` r
rename_child(x, i, p)
```

## Arguments

- x:

  The child element to be renamed.

- i:

  The index or name of the child element.

- p:

  The name of the parent element.

## Value

A renamed list of child elements.
